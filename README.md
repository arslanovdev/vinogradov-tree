# Генеалогическое древо семьи Виноградовых

Интерактивное древо: Фёдоровский район, Башкортостан. Точка отсчёта — Виноградова Надежда Дмитриевна.

**Открыть:** https://arslanovdev.github.io/vinogradov-tree/

## Как устроено

- `fedorovka_family.ged` — **единственный источник данных** (GEDCOM 5.5.1). Всё древо строится из него.
- `index.html` — приложение. При открытии оно само подгружает `fedorovka_family.ged` (через `fetch`) и рисует древо. Сам `index.html` менять не нужно.

## Как обновить древо

1. Отредактируйте `fedorovka_family.ged` (добавьте/исправьте персон, семьи, заметки).
2. Закоммитьте и запушьте:
   ```bash
   git add fedorovka_family.ged
   git commit -m "обновление данных"
   git push
   ```
3. Через ~1 минуту GitHub Pages пересоберётся, и древо на сайте обновится. Менять `index.html` не требуется.

## Разработка приложения

Если нужно поменять сам вид/логику древа, исходник лежит в рабочей папке (`../vinogradka/template.html`),
сборка — `python3 ../vinogradka/build.py`. Скрипт делает две версии:
- автономную (`Древо семьи Виноградовых.html`, GEDCOM зашит внутрь — открывается двойным кликом офлайн);
- для Pages (`index.html`, GEDCOM грузится через fetch).

## Формат данных

Каждая персона размечена точностью `[ТОЧНОСТЬ A/B/C]`; архивные подтверждения — в заметках `[ПН]`
(«Память народа»). Цвета линий: отец / мать / вы. Метки: ✦ военная служба, ✝ погиб/умер.
